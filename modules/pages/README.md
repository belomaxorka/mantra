# Pages Module

Модуль для интеграции опубликованных страниц в навигационное меню сайта.

## Описание

Этот модуль автоматически добавляет опубликованные страницы с флагом `show_in_navigation` в главное меню сайта через систему хуков.

## Возможности

- Автоматическое добавление страниц в навигацию
- Фильтрация по статусу `published`
- Фильтрация по флагу `show_in_navigation`
- Настраиваемый порядок отображения через `navigation_order`

## Как это работает

Модуль подключается к хуку `theme.navigation` и добавляет элементы меню для всех страниц, которые:
1. Имеют статус `published`
2. Имеют флаг `show_in_navigation: true`

Каждая страница добавляется с:
- **ID**: `page-{slug}`
- **Title**: заголовок страницы
- **URL**: `/{slug}`
- **Order**: значение из поля `navigation_order` (по умолчанию 50)

## Установка

1. Модуль должен быть включен в `content/settings/config.json`:

```json
{
  "modules": {
    "enabled": ["admin", "pages", ...]
  }
}
```

2. Модуль загружается автоматически при старте приложения

## Управление страницами

Для управления страницами используйте админ-панель:
- `/admin/pages` - список страниц
- Включите чекбокс "Show in navigation menu" при создании/редактировании страницы
- Настройте порядок через поле "Navigation order" (меньшее число = выше в меню)

## Связанные модули

- `admin-modules/pages` - админ-панель для управления страницами
- Этот модуль (`pages`) - публичная интеграция в навигацию

## Технические детали

- **Хук**: `theme.navigation`
- **Коллекция**: `pages`
- **Приоритет**: стандартный (10)
