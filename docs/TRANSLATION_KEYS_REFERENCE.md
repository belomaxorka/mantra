## Usage Pattern

All keys follow the pattern: `module_id.category.key`

### Examples:
```php
// Universal actions from admin module
<?php echo t('admin.common.save'); ?>
<?php echo t('admin.common.cancel'); ?>
<?php echo t('admin.common.delete'); ?>

// Module-specific translations
<?php echo t('admin.pages.title'); ?>
<?php echo t('admin.posts.new_post'); ?>
<?php echo t('admin.dashboard.welcome'); ?>

// Sidebar groups
<?php echo t('admin.sidebar.group.content'); ?>
<?php echo t('admin.sidebar.group.system'); ?>
```

### In Module Registration:
```php
$this->registerSidebarItem(array(
    'id' => 'pages',
    'title' => array('key' => 'admin.pages.title', 'fallback' => 'Pages'),
    'group' => array('key' => 'admin.sidebar.group.content', 'fallback' => 'Content'),
));
```
